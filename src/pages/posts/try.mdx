<div class="sim-article">
  <h2 id="overview">What this simulation shows</h2>
  <p>
    This demo illustrates a basic 3D scene containing a rotating cube. It's a minimal yet interactive introduction to WebGL rendering using Three.js.
  </p>

  <h3 id="how-it-works">How it works</h3>
  <p>
    A React component (`SimpleSim.jsx`) is imported and hydrated solely on the client. It sets up a Three.js scene, a perspective camera, and animates the cube in a render loop.
  </p>

  <h3 id="rendering-details">Rendering details</h3>
  <ul>
    <li><strong>Cube geometry:</strong> Standard Three.js BoxGeometry with a MeshNormalMaterial.</li>
    <li><strong>Animation loop:</strong> Rotates the cube continuously using `requestAnimationFrame`.</li>
    <li><strong>Responsive canvas:</strong> Resizes with the container for proper aspect ratio.</li>
  </ul>

  <h3 id="purpose">Why this matters</h3>
  <ul>
    <li>Shows how WebGL simulations can live inside Astro MDX pages via React islands.</li>
    <li>Perfect starting point for expanding into more complex simulations.</li>
    <li>Maintains performance by isolating interactive code in a lightweight client-only component.</li>
  </ul>
</div>
